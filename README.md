# MS Pacman キーボード操作ゲーム

このプログラムは、OpenAI Gymの`MsPacman-v0`環境をキーボードで操作することができるゲームです。

## 機能

- 矢印キー（上・右・左・下）でパックマンを操作
- キー入力があるまで待機し、入力があるたびに画面を更新
- ゲームオーバー時にスコアを表示

## 必要条件

- Python 3.8以上
- gymnasium
- pygame

## インストール方法

```bash
# uvを使用して環境を作成
uv init

# 必要なライブラリをインストール
uv add gymnasium pygame
```

## 使い方

```bash
python game.py
```

## 操作方法

- 矢印キー上: パックマンを上に移動
- 矢印キー右: パックマンを右に移動
- 矢印キー左: パックマンを左に移動
- 矢印キー下: パックマンを下に移動
- ESCキー: ゲーム終了
